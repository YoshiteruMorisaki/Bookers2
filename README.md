# DMM WEBCAMP アプリケーションを作成しよう2

3つ目の提出課題であるBookers2のリポジトリです。

---

# Bookers2

Bookers同様に本の感想を投稿できるアプリです。

---

# 機能

- 標準認証機能によるサインアップ、ログイン・ログアウト
- 新規投稿および投稿内容の編集・削除
- 登録ユーザーによる投稿内容の一覧とその詳細画面
- プロフィール画像を含むユーザーの詳細設定

---

# 使用技術

- Ruby on Rails 8
- Bootstrap 5
- Active Storage
- RSpec
